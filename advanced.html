API Endpoints

GET
/api/:date?
Primary temporal conversion endpoint. Accepts Unix timestamps or date strings. Returns JSON with unix and utc properties.
{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}
GET
/api/1451001600000
Direct Unix timestamp input. Milliseconds since January 1, 1970. Returns corresponding UTC coordinate.
{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}
GET
/api/
Empty parameter returns current server time. Useful for synchronization and heartbeat monitoring.
{
  "unix": 1704067200000,
  "utc": "Mon, 01 Jan 2024 00:00:00 GMT"
}
GET
/api/invalid-date
Error handling demonstration. Invalid inputs return structured error object with descriptive messaging.
{
  "error": "Invalid Date"
}
